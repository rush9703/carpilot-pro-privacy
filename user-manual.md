## <a id="getting-started"></a>1. Getting Started

### Mounting Your Phone
For accurate data, your iPhone must be securely mounted in the vehicle.
1.  **Secure Mount**: Use a sturdy phone holder. Avoid loose areas.
2.  **Orientation**: Ensure the phone is aligned with the vehicle's direction of travel.
    *   Go to **Settings > Phone Mounting** to calibrate.
    *   **Important**: If mounting vertically (screen facing driver), tilt the device at least 30Â° back to avoid sensor issues.

### Permissions
CarPilot requires the following permissions:
- **Location**: "Always Allow" or "While Using App" (for high-precision GPS).
- **Motion & Fitness**: For accelerometer and gyroscope data.
- **Microphone** (Optional): For noise level analysis (recorded data is sound level only, no voice).

## <a id="performance-testing"></a>3. Performance Testing

Measure your car's straight-line capabilities.

### <a id="acceleration-tests"></a>Acceleration Tests
*   **0-100 km/h (0-60 mph)**: Measures time to reach speed.
*   **0-400m (1/4 mile)**: Measures time and trap speed over a distance.
*   **Procedure**:
    1.  Select **Performance Test** (Orange button).
    2.  Choose your test type.
    3.  Come to a complete stop.
    4.  The app will say "Ready".
    5.  Launch! The app auto-starts timing when movement determines.

### <a id="braking-tests"></a>Braking Tests
*   **100-0 km/h (60-0 mph)**: Measures stopping distance and time.
*   **Procedure**:
    1.  Select **Braking Test**.
    2.  Accelerate above the target speed (e.g., 105 km/h).
    3.  Brake hard to a complete stop.

### <a id="custom-tests"></a>Custom Tests
You can define your own speed ranges (e.g., 60-120 km/h) or distances in the test configuration menu.

## <a id="track-recording"></a>4. Track Recording

Record lap times, racing lines, and sector analysis.

### <a id="automatic-detection"></a>Automatic Detection
If you are near a supported track, a **Track Card** appears on the Home Screen. Tap **"Use Track"** to start.

### <a id="manual-selection"></a>Manual Selection
1.  Tap **Track Recording** (Blue button).
2.  Browse the Track Library to find your circuit.

### <a id="creating-custom-tracks"></a>Creating Custom Tracks
1.  Go to **Data > Tracks**.
2.  Tap **+ (Create Track)**.
3.  Select a previous driving session.
4.  Scrub the timeline to find the Start/Finish line point.
5.  Set the start line orientation.
6.  Save custom track.

## <a id="analysis--reports"></a>5. Analysis & Reports

### <a id="performance-analysis"></a>Performance Analysis
View detailed graphs for acceleration and braking runs.
*   **Charts**: Speed vs Time, G-Force vs Time.
*   **Metrics**: Peak G, 0-60 time, 60ft time, slope/gradient info.

### <a id="track-analysis"></a>Track Analysis
Compare laps and analyze driving lines.
*   **Lap Timer**: View all lap times.
*   **Trajectory Map**: Color-coded heat map of speed or G-force.
*   **Compare**: Select two laps to overlay speed traces and see where you gained/lost time.

### <a id="comfort--noise-analysis"></a>Comfort & Noise Analysis
*   **Comfort**: Analyzes vertical G-force and vibrations (Ride Quality).
*   **Noise**: Maps interior noise levels (dB) against vehicle speed.

## <a id="data-management"></a>6. Data Management

### <a id="importexport-vbo"></a>Import/Export
CarPilot supports the industry-standard **VBO** format (Racelogic VBOX).
*   **Export**: Share any session as a VBO file to analyze in circuit tools like Circuit Tools.
*   **Import**: Open `.vbo` files from other devices to analyze in CarPilot.

### <a id="session-library"></a>Session Library
Access all your history under the **Data** tab.
*   **Filter**: By Date, Track, or Test Type.
*   **Delete**: Swipe left on any session to delete.

## <a id="settings"></a>7. Settings

Configure the app to your needs.
*   **Language**: Change app language.
*   **Units**: Switch between Metric (km/h, m) and Imperial (mph, ft).
*   **Video**: Configure resolution (1080p/4K) and camera selection.
*   **Drivers & Vehicles**: Manage profiles for different cars and drivers.
*   **Screen Recording**: Toggle simultaneous screen recording during tests (with optional mic audio).
